Chaconne 2 Quick Start

The opening page looks a bit empty at first, but once you have created some scores, they will be available here as thumbnails.

To get started, tap on ‘New Score’ - the dialog to create a new score isn’t too different from the old app, with a few extra options (setting the number of bars, allowing an upbeat bar).

Keyboard Input
 
Keyboard Input

The keyboard input is similar to the old app in many ways. You still tap in the notes, enter rests, change the duration using the scrolling selection, set the accidental and number of dots, use the ‘hold’ button to create chords etc.

The main differences are:

DELETE – the delete button is permanently available. When using the piano keyboard, it will delete whatever is at the marker. No more having to change to a separate ‘delete’ mode and zoom in to tap on the chord!

LONG PRESS – Long pressing on a segment of the score selects it – tapping again on another segment will select a whole range. You then have a whole range of features available:

   

 

 

 

COPY/CUT/PASTE  - Use the floating layout that appears in the top right corner. Press on ‘copy’(left) to copy the selection to the clipboard. To paste, long-press again on the target segment, and press the paste (right) button. (If these buttons get in the way, you can long press them and move them somewhere else).

DELETE SELECTION – When in this state the eraser button will delete all events in the selection.

DELETE BARS – Long Press the delete button to delete all selected bars (not just their contents).

EDITING NOTES – The up-down arrows on the left move all notes in the selection up and down -  if the 8va button is selected they will move them a whole octave at once

BEAMING/SEPARATING NOTES – The beam symbols in the middle, respectively, beam notes together, separate them into individual tailed notes, and restore them to their default beaming. (Assuming of course they are 8th notes or smaller).

CHANGING STEM DIRECTION – The up/down stem buttons on the right change the stem direction of the selected range – the cross button restores them to their defaults (need better icons..)

CHANGING DURATION – This allows you to change the direction of selected notes

 

But what if I only want to edit one note in a chord? The cycle button (round arrows) changes the selection to individual events – notes in a chord, and anything else that happens to be at the start of the selected area (clef, text etc). To select one note, select the entire segment, then use this button to cycle through the notes in the chord – you can then move it up or down or delete it.

 

CLEAR SELECTION – To get rid of the selection and restore the keyboard, just tap the ‘X’ button on the very bottom row.

RESTORE KEYBOARD WHILE RETAINING THE SELECTION  - This is achieved by tapping the keyboard button under the cycle button. But why would you want to? Whilst a range is selected, anything done in the keyboard panel is repeated across the entire range. In this way you can repeat the same chord 400 times if you want.
